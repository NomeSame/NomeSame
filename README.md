### Hi 👋 I'm NomeSame

I build software products by pairing my own ideas and judgment with AI-assisted development.

**How I see it:** in 2026, *writing* the code is the part machines already do faster and cleaner than Humans could. The parts that still genuinely need a human, and the parts I'm actually good at, are deciding **what's worth building**, shaping **how it should feel to use**, and **testing it until it holds up**. That's where I put my energy, because that's where the lasting value is. A feature that ships is only ever as good as the verification behind it.

So I work idea-first: I design the product, direct the implementation, and own the testing. I treat AI as the fastest tool I've ever had, not as a replacement for taste, product sense, or accountability.
